-----------------------------------
Usage
-----------------------------------

To access the API for:
-user registration
-play tictactoe
-heartbeat application

the base URL is:
http://<host>:<port>/webmain/api/

you can then check the controllers for request URLs, for example:
http://localhost:8080/webmain/api/status
http://localhost:8080/webmain/api/startGame
http://localhost:8080/webmain/api/move/5

You can use Chrome Postman to make the request and see the response

-----------------------------------

To view the webGL demo, use an internet browser and the base URL:
http://localhost:8080/webmain/webgl/

For example: 
http://localhost:8080/webmain/webgl/a.html
